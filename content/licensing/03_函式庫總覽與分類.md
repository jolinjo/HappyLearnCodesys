# CODESYS 函式庫總覽與分類

> 資料來源：CODESYS 官網、[helpme-codesys.com](https://content.helpme-codesys.com/) 函式庫文件。  
> 以下依**通訊、軸控/運動、檔案、時間、系統**等大類整理，並列出主要 Function / Function Block，便於查閱。

---

## 一、通訊類（Communication）

### 1.1 SysCom（序列埠 RS232）

提供序列介面（RS232）存取，用於與週邊設備、條碼機、印表機等通訊。

| 類型 | 名稱 | 說明 |
|------|------|------|
| **Function** | `SysComOpen` | 開啟序列埠連線（舊版介面） |
| **Function** | `SysComOpen2` | 開啟序列埠，指定埠號與參數 |
| **Function** | `SysComOpen3` | 以設定結構開啟序列埠，傳回 handle |
| **Function** | `SysComClose` | 關閉序列埠連線 |
| **Function** | `SysComRead` | 自序列埠讀取資料 |
| **Function** | `SysComWrite` | 寫入資料至序列埠 |
| **Function** | `SysComSetSettings` | 設定介面參數（相容用） |
| **Function** | `SysComSetSettings2` | 設定介面參數（handle + 設定） |
| **Function** | `SysComGetSettings` | 取得目前介面設定 |
| **Function** | `SysComGetSettings2` | 取得介面設定（新版） |
| **Function** | `SysComSetTimeout` | 設定讀寫逾時 |
| **Function** | `SysComPurge` | 清除緩衝區 |

**列舉/結構**：鮑率、同位、停止位元、DTR/RTS、逾時、埠號等（如 `SysComSettings`）。

- 文件：[SysCom Library](https://content.helpme-codesys.com/en/libs/SysCom/Current/index.html)

---

### 1.2 其他通訊相關

- **OPC UA Server**：CODESYS 內建 OPC UA 伺服器，可於專案中啟用。
- **Ethernet / Fieldbus**：EtherCAT、PROFINET、EtherNet/IP、Modbus TCP/Serial、CANopen、PROFIBUS、J1939 等，依設備與授權透過對應 device/stack 使用，非單一「函式庫」名稱。
- **Communication Manager / Symbol Configuration**：需 **Communication M / XXL** 授權，用於標籤交換與連線設定。

---

## 二、軸控 / 運動類（Motion Control）

### 2.1 SM3_Basic（SoftMotion 基礎）

SoftMotion V3 必備基礎庫，提供 PLCopen 運動控制 FB 與驅動介面。需搭配 SoftMotion 授權使用。

#### 驅動介面（DriveInterface）

| 類型 | 名稱 | 說明 |
|------|------|------|
| **FB** | `AXIS_REF_SM3` | 單軸驅動介面，連接實體驅動、狀態與週期任務 |
| **FB** | `AXIS_REF_VIRTUAL_SM3` | 虛擬軸介面 |
| **FB** | `AXIS_REF_MAPPING_SM3` | 軸映射介面 |

#### 管理/設定（Administrative / Configuration）

| 類型 | 名稱 | 說明 |
|------|------|------|
| **FB** | `MC_Power` | 軸使能/關閉 |
| **FB** | `MC_Reset` | 清除軸錯誤 |
| **FB** | `MC_SetPosition` | 設定目前位置 |
| **FB** | `SMC_SetSoftwareLimits` | 設定軟體行程限制 |
| **FB** | `SMC_SetRampType` | 設定加減速類型 |
| **FB** | `SMC_SetMovementType` | 設定運動類型 |
| **FB** | `SMC_SetCustomRampType` | 自訂加減速曲線 |
| **FB** | `SMC_SetControllerMode` | 設定控制器模式 |
| **FB** | `SMC_SetAdditionalConversionFactors` | 額外單位換算係數 |
| **FB** | `SMC_ChangeGearingRatio` | 變更電子齒輪比 |
| **FB** | `SMC_ChangeDynamicLimits` | 變更速度/加減速限制 |
| **FB** | `SMC3_PersistPosition` / `SMC3_PersistPositionSingleturn` / `SMC3_PersistPositionLogical` | 位置持久化 |
| **FB** | `SMC3_BrakeControl` | 煞車控制 |

#### 單軸運動（Movement - PLCopen）

| 類型 | 名稱 | 說明 |
|------|------|------|
| **FB** | `MC_MoveAbsolute` | 絕對位置定位 |
| **FB** | `MC_MoveRelative` | 相對位移 |
| **FB** | `MC_MoveVelocity` | 速度模式運動 |
| **FB** | `MC_MoveAdditive` | 附加位移 |
| **FB** | `MC_MoveSuperImposed` | 疊加運動 |
| **FB** | `MC_Halt` | 減速停止 |
| **FB** | `MC_Stop` | 立即停止 |
| **FB** | `MC_Home` | 回原點 |
| **FB** | `MC_Jog` | 點動 |
| **FB** | `MC_SetOverride` | 設定速度 override |
| **FB** | `MC_AccelerationProfile` / `MC_VelocityProfile` / `MC_PositionProfile` | 輪廓運動 |

#### 主從/電子凸輪（Master/Slave）

| 類型 | 名稱 | 說明 |
|------|------|------|
| **FB** | `MC_GearIn` | 電子齒輪嚙合 |
| **FB** | `MC_GearInPos` | 定位後齒輪嚙合 |
| **FB** | `MC_GearOut` | 電子齒輪脫開 |
| **FB** | `MC_CamIn` | 凸輪嚙合 |
| **FB** | `MC_CamOut` | 凸輪脫開 |
| **FB** | `MC_Phasing` | 相位調整 |

#### 直接控制 / 追隨（Direct）

| 類型 | 名稱 | 說明 |
|------|------|------|
| **FB** | `SMC_FollowPosition` | 位置跟隨 |
| **FB** | `SMC_FollowVelocity` | 速度跟隨 |
| **FB** | `SMC_FollowPositionVelocity` | 位置+速度跟隨 |
| **FB** | `SMC_FollowSetValues` | 設定值跟隨 |
| **FB** | `SMC_SetTorque` | 轉矩設定 |
| **FB** | `SMC_TrackAxis` | 軸追蹤 |
| **FB** | `SMC_TrackSetValues` | 設定值追蹤 |

#### 參數存取（Parameter access）

| 類型 | 名稱 | 說明 |
|------|------|------|
| **FB** | `MC_ReadParameter` / `MC_ReadBoolParameter` | 讀取驅動參數 |
| **FB** | `MC_WriteParameter` / `MC_WriteBoolParameter` | 寫入驅動參數 |
| **Function** | `SMC_ParameterNumber_CoE` / `SMC_ParameterNumber_SoE` | CoE/SoE 參數編號 |

#### CAM

| 類型 | 名稱 | 說明 |
|------|------|------|
| **FB** | `SMC_CAM_ObjectManager` | CAM 物件管理 |
| **FB** | `SMC_CamEditor` | CAM 編輯 |
| **FB** | `SMC_ReadCAM` / `SMC_WriteCAM` | 讀寫 CAM 檔 |

#### 診斷

| 類型 | 名稱 | 說明 |
|------|------|------|
| **FB** | `SMC_ReadFBError` | 讀取 FB 錯誤 |
| **Function** | `SMC_ClearFBError` | 清除 FB 錯誤 |
| **FB** | `SMC_PerfStat` / `SMC_SeriesStat` | 效能統計 |
| **Function** | `SMC_PerfTimerSum` | 效能計時加總 |
| **FB** | `SMC_AxisDiagnosticLog` | 軸診斷紀錄 |
| **FB** | `MC_DigitalCamSwitch` | 數位凸輪開關 |

#### 輔助

| 類型 | 名稱 | 說明 |
|------|------|------|
| **Function** | `SMC_BrakingDistance` | 計算煞車距離 |

- 文件：[SM3_Basic](https://content.helpme-codesys.com/en/libs/SM3_Basic/4.14.0.0/index.html)

---

### 2.2 SM3_CNC

提供 G-code（DIN 66025）解讀與插補，用於 CNC 與路徑控制。內含讀取 NC 檔、路徑處理與插補用 FB，需搭配 SoftMotion CNC/軸組授權。

---

### 2.3 其他運動相關

- **SM3_ Robotics**：機器人運動學與路徑規劃（若專案有安裝）。
- 各廠 **AXIS_REF_*** 擴充庫：對應 EtherCAT、PROFINET 等驅動，延伸 `AXIS_REF_SM3`。

---

## 三、檔案類（File）

### 3.1 SysFile（系統底層檔案）

提供 Runtime 底層檔案操作，同步、低階 API。

| 類型 | 名稱 | 說明 |
|------|------|------|
| **Function** | `SysFileOpen` | 開啟檔案 |
| **Function** | `SysFileClose` | 關閉檔案 |
| **Function** | `SysFileRead` | 讀取 |
| **Function** | `SysFileWrite` | 寫入 |
| **Function** | `SysFileSetPos` | 設定讀寫位置 |
| **Function** | `SysFileGetPos` | 取得目前位置 |
| **Function** | `SysFileGetSize` / `SysFileGetSizeByHandle` | 取得檔案大小 |
| **Function** | `SysFileCopy` | 複製檔案 |
| **Function** | `SysFileDelete` / `SysFileDeleteByHandle` | 刪除檔案 |
| **Function** | `SysFileRename` | 重新命名 |
| **Function** | `SysFileTruncate` | 截斷檔案 |
| **Function** | `SysFileFlush` | 寫入緩衝區 |
| **Function** | `SysFileEOF` | 是否到達檔尾 |
| **Function** | `SysFileGetName` / `SysFileGetName2` | 取得檔名 |
| **Function** | `SysFileGetPath` | 取得路徑 |
| **Function** | `SysFileGetStatus` / `SysFileGetStatus2` | 取得狀態 |
| **Function** | `SysFileGetTime` | 取得檔案時間 |

**列舉/結構**：`ACCESS_MODE`、`SYS_FILE_STATUS`、`SYS_FILETIME` 等。

- 文件：[SysFile](https://content.helpme-codesys.com/en/libs/SysFile/Current/index.html)

---

### 3.2 SysFileAsync（非同步檔案）

非同步檔案 I/O，避免阻塞 PLC 週期，介面與 SysFile 類似但為非同步呼叫。

- 文件：[SysFileAsync](https://content.helpme-codesys.com/en/libs/SysFileAsync/Current/index.html)

---

### 3.3 CAA File（CAA 標準檔案與目錄）

CAA 標準化檔案與目錄存取，支援本機路徑、FTP、HTTP 等。僅支援二進位模式。

#### 檔案（File）

| 類型 | 名稱 | 說明 |
|------|------|------|
| **FB** | `FILE.Open` | 開啟檔案 |
| **FB** | `FILE.Close` | 關閉檔案 |
| **FB** | `FILE.Read` | 讀取 |
| **FB** | `FILE.Write` | 寫入 |
| **FB** | `FILE.SetPos` | 設定位置 |
| **FB** | `FILE.GetPos` | 取得位置 |
| **FB** | `FILE.GetSize` | 取得大小 |
| **FB** | `FILE.GetTime` | 取得時間 |
| **FB** | `FILE.GetAttribute` | 取得屬性 |
| **FB** | `FILE.Copy` | 複製 |
| **FB** | `FILE.Delete` | 刪除 |
| **FB** | `FILE.Rename` | 重新命名 |
| **FB** | `FILE.Flush` | 寫入緩衝區 |
| **FB** | `FILE.EOF` | 是否檔尾 |

#### 目錄（Directory）

| 類型 | 名稱 | 說明 |
|------|------|------|
| **FB** | `DirOpen` | 開啟目錄 |
| **FB** | `DirClose` | 關閉目錄 |
| **FB** | `DirList` | 列出目錄內容 |
| **FB** | `DirCreate` | 建立目錄 |
| **FB** | `DirCopy` | 複製目錄 |
| **FB** | `DirRemove` | 刪除目錄 |
| **FB** | `DirRename` | 重新命名目錄 |

#### 函式

| 類型 | 名稱 | 說明 |
|------|------|------|
| **Function** | `FILE.GetProperty` | 查詢實作能力（如是否支援 FTP/HTTP） |

路徑可為本機絕對/相對、`file://`、`ftp://`、`http://` 等（依實作而定）。

- 文件：[CAA File](https://content.helpme-codesys.com/en/libs/CAA%20File/Current/index.html)

## 四、時間類（Date / Time）

### 4.1 SysTimeRtc（即時時鐘）

存取目標系統 RTC（日期/時間）。

| 類型 | 名稱 | 說明 |
|------|------|------|
| **Function** | `SysTimeRtcGet` | 取得目前日期時間 |
| **Function** | `SysTimeRtcSet` | 設定 RTC |
| **Function** | `SysTimeRtcConvertDateToUtc` / `SysTimeRtcConvertUtcToDate` | 日期與 UTC 轉換 |
| **Function** | `SysTimeRtcConvertLocalToUtc` / `SysTimeRtcConvertUtcToLocal` | 本地與 UTC 轉換 |
| **Function** | `SysTimeRtcGetTimezone` / `SysTimeRtcSetTimezone` | 時區取得/設定 |
| **Function** | `SysTimeRtcControl` | RTC 控制 |
| **Function** | `SysTimeRtcHighResGet` / `SysTimeRtcHighResSet` | 高解析度時鐘 |
| **Function** | `SysTimeRtcConvertDateToHighRes` / `SysTimeRtcConvertHighResToDate` | 與高解析度時間轉換 |

**結構**：`RTS_SYSTIMEDATE`（年/月/日/時/分/秒/毫秒/星期/一年中第幾天等）。

- 文件：[SysTimeRtc](https://content.helpme-codesys.com/en/libs/SysTimeRtc/3.5.17.0/index.html)

---

### 4.2 SysTimeCore（高解析度時間間隔）

取得高解析度時間戳或經過時間（毫秒/微秒/奈秒等級），用於量測與計時。

| 類型 | 名稱 | 說明 |
|------|------|------|
| **Function** | `SysTimeGetMs` | 毫秒解析度 |
| **Function** | `SysTimeGetUs` | 微秒解析度 |
| **Function** | `SysTimeGetNs` | 奈秒解析度（依平台） |

（實際名稱以各版 SysTimeCore 文件為準。）

## 六、分類總表（依用途）

| 大類 | 函式庫 | 主要用途 |
|------|--------|----------|
| **通訊** | SysCom | 序列埠 RS232 |
| **通訊** | OPC UA / Fieldbus 設備 | 乙太網路、現場匯流排（依設備與授權） |
| **軸控/運動** | SM3_Basic | 單軸/主從/凸輪/齒輪、驅動介面 |
| **軸控/運動** | SM3_CNC | G-code、CNC 插補 |
| **軸控/運動** | SM3_ Robotics | 機器人（若已安裝） |
| **檔案** | SysFile | 同步、底層檔案 I/O |
| **檔案** | SysFileAsync | 非同步檔案 I/O |
| **檔案** | CAA File | 標準化檔案/目錄、FTP/HTTP |
| **時間** | SysTimeRtc | RTC 日期時間、時區 |
| **時間** | SysTimeCore | 高解析度時間/間隔 |
| **系統** | 各庫 Project/Library Information | 版本、標題、屬性查詢 |
